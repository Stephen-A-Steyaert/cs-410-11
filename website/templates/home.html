How This Works

This template demonstrates Jinja2 template inheritance:

  • {% raw %}{{ url_for('blueprint.home') }}{% endraw %} - Sets the HREF, useful for anchor tags!
  • {% raw %}{{ url_for('development_example.example') }}{% endraw %} - Goes to the Example Route
  • {% raw %}{% block title %}{% endblock %}{% endraw %} - Sets the page title
  • {% raw %}{% block content %}{% endblock %}{% endraw %} - Fills in the main content area
